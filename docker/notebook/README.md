# Use notebook

## 1. Setup environment

### 1.1. Install python virtualenv

```bash
$ python -m venv .venv --prompt='study-devops-docker'
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

### 1.2. Install bash extention

```bash
$ python -m bash_kernel.install
```

## 2. Start

```bash
$ jupyter lab
```
